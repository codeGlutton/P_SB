1. 데이터 기록이 시작되는 첫 행은 PrefixId 와 숫자 로 기록되야한다.
-> 해당 숫자는 같은 id번호를 공유하는 데이터 형의 분별용 1000단위 번호이다. ex) 1000, 2000, 3000, ... 최대 64000

2. PrefixId 행 이후, 등장하는 행은 순서대로 열의 타입형을 기록한 행과 열 별칭을 기록한 행이다.
-> 필수 열 별칭으로 0번째 행은 Id, 1번째 행은 ServerName, 2번째 행은 ClientName, 3번째 행은 ClientBasePath이다.
-> 타입형의 경우, 첫 글자만 대문자로 기록할 것. <>내부는 대소문자를 구분해줘야한다.
-> Server가 별칭에 prefix로 붙으면 서버측만 보이며, Client가 별칭에 prefix로 붙은면 클라측에만 보인다.

3. 이후 등장하는 행은 모두 데이터이다.
-> 문자열, 숫자 모두 그냥 기록할 것
